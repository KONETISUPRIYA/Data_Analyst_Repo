# Bank Customer & Transaction Analysis using Oracle SQL

## Project Overview

This project demonstrates the design and analysis of a banking database using Oracle SQL. The database consists of Customers, Accounts, and Transactions tables, enabling analysis of customer behavior, account balances, branch performance, and transaction activity.

## Objective

To perform business-oriented analysis on banking data using SQL and generate insights through data querying and reporting techniques.

## Database Schema

### Customers

* Customer ID
* Customer Name
* City
* Account Type

### Accounts

* Account ID
* Customer ID
* Balance
* Branch

### Transactions

* Transaction ID
* Account ID
* Transaction Date
* Transaction Amount
* Transaction Type

* INNER JOIN
* Multi-table Joins

## Analysis Performed

### Customer Analysis

* Total Customers
* Customer Account Information
* Customer Balance Analysis

### Branch Analysis

* Branch-wise Customer Count
* Branch-wise Total Balance
* Highest Performing Branch

### Transaction Analysis

* Deposit vs Withdrawal Count
* Deposit vs Withdrawal Amount
* Customer Transaction History

## Key Insights

* Total Customers: 10
* Total Bank Balance: ₹664,000
* Average Customer Balance: ₹66,400
* Highest Balance: ₹92,000
* Mumbai Branch has the highest total balance.
* Deposits exceed withdrawals by ₹30,000.
